# Rasmlar

Bu papkaga quyidagi rasmlarni qo'ying:

1. **profile.jpg** - Sizning rasmingiz (200x200px tavsiya etiladi)
   - Yumaloq qilib ko'rsatiladi
   - Screenshot va kattalashtirishdan himoyalangan

2. **quyosh24.jpg** - Quyosh paneli rasmi (agar kerak bo'lsa)

## Rasm qo'shish:
1. Rasmni `images` papkasiga nusxalang
2. Nomini `profile.jpg` qilib o'zgartiring
3. Git ga qo'shing: `git add images/profile.jpg`
4. Commit qiling: `git commit -m "Profile rasm qo'shildi"`
5. Push qiling: `git push`
6. Vercel avtomatik yangilanadi!
